# Manipulator-Model-1
This is a basic Manipulator project.

This repository documents the initial physical prototype phase

Specifications & Hardware
•	Controller: Arduino Uno R3
-   Transmiter/Reciever: HC-05
•	Actuation: 3x Servo Servos MG996R (Base Yaw, Shoulder Pitch, Elbow Pitch)
•	Structure: Custom laser-cut acrylic chassis (DXF files included)
•	Original Control Scheme: Custom serial-parse protocol accepting raw joint-space instructions (e.g., M1123 to drive Motor 1 to 123 degrees‭‬).
